# Quick Reference: Referral Codes for Staff & Admin

## What is a Referral Code?

Your **referral code** is a unique identifier that applicants (pelamar) must enter when they register in the mobile app. This helps track which staff member recruited each applicant.

## Your Referral Code Format

- **Staff**: `S-XXXXXX` (Example: `S-ABC123`)
- **Admin**: `A-XXXXXX` (Example: `A-XYZ789`)

## How to Find Your Code

### Option 1: Admin Dashboard

1. Login to admin dashboard: https://data.kms-connect.com/admin/
2. Click on your profile/account
3. Your referral code is displayed in the "Kode Rujukan" field
4. Copy the code

### Option 2: Ask Your Administrator

If you don't have access to the admin dashboard, ask your administrator to provide your referral code.

## Tracking Your Referrals

### View Your Referrals in Admin Dashboard

1. Login to admin dashboard
2. Navigate to "Daftar Profil Pelamar" (Applicant Profiles)
3. Filter by "Perujuk" = Your email
4. You'll see all applicants you've referred

### Check Your Performance

Your supervisor/admin can generate reports showing:
- Total applicants you've referred
- How many are verified/accepted
- Comparison with other staff members

## Troubleshooting

### Problem: "I don't have a referral code yet"

**Solution:**
1. Contact your administrator
2. They will generate a code for you
3. Or run command: `python manage.py generate_referral_codes`

### Problem: "My code doesn't work"

**Checks:**
1. Is your account active? (Check with admin)
2. Are you entering it correctly? (Check spelling)
3. Contact technical support if issue persists

### Problem: "I want to change my code"

**Process:**
1. Login to admin dashboard
2. Edit your user profile
3. Change "Kode Rujukan" field
4. Save changes
5. **Important**: Inform all pending applicants of new code
